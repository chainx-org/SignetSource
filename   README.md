# Get the signet address

## Linux

Download the file [start.sh](./Linux/start.sh) 

~~~sh
curl https://raw.githubusercontent.com/chainx-org/threshold_signature_api/main/signet/Linux/start.sh -o start.sh
~~~

Start up

~~~sh
sh start.sh
~~~
